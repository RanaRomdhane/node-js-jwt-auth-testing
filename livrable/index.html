Pourquoi ces outils ?

Jest offre un environnement de test complet avec mocking natif et snapshots. Supertest permet de tester les endpoints HTTP sans démarrer le serveur. GitHub Actions assure l'automatisation gratuite pour les projets open-source. SonarQube détecte les code smells, bugs et vulnérabilités de sécurité.

Utilité du CI/CD

Chaque push déclenche automatiquement: installation des dépendances, analyse ESLint, démarrage MongoDB, exécution de tous les tests avec rapport de couverture. Feedback en moins de 5 minutes! Garantit qu'aucune régression n'est introduite.

Tests Unitaires

auth.unit.test.js
describe("UT-01: Hashage Password", () => {
test("Doit hasher correctement", () => {
const password = "SecureP@ss123";
const hashed = bcrypt.hashSync(password, 8);
expect(hashed).not.toBe(password);
expect(bcrypt.compareSync(
password, hashed
)).toBe(true);
});
});
